## Expiry Tracking

| Category | Default Shelf Life |
|----------|-------------------|
| Leafy Vegetables | 3 days |
| Other Vegetables | 7 days |
| Fruits | 5 days |
| Dairy (Milk, Curd) | 5 days |
| Dairy (Paneer, Cheese) | 7 days |
| Eggs | 14 days |
| Bread | 5 days |
| Grains & Pulses | No expiry |
| Spices | No expiry |
