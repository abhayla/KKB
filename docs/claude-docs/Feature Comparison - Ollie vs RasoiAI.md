# Feature Comparison: Ollie.ai vs RasoiAI

## Summary

### Features RasoiAI Adds (vs Ollie) - India-Specific Value

| Feature | Description |
|---------|-------------|
| **Festival Intelligence** | 30+ Indian festivals with auto-suggested menus |
| **Fasting Mode** | Navratri, Ekadashi, Shravan - religious practices |
| **Regional Cuisine Zones** | 4-zone depth (N/S/E/W) with authentic recipes |
| **Jain/Sattvic Diets** | No root vegetables, no onion-garlic support |
| **Elder-Friendly Tags** | Joint family support - low-oil, soft, diabetic |
| **Mixed Diet Families** | Veg + Non-veg options in same meal plan |
| **WhatsApp Grocery** | India's shopping pattern - share to kirana |
| **Offline Mode** | Tier 2-3 city connectivity issues |
| **Hindi Language** | Bilingual UI + recipe content |
| **Seasonal Produce** | Mango season, monsoon foods, winter warmers |
| **Indian Measurements** | Katori, chammach alongside metric |

### Features RasoiAI Skips (vs Ollie)

| Feature | Rationale |
|---------|-----------|
| **iOS App** | 95% India is Android |
| **Instacart/Amazon/Walmart Integration** | Not available in India (WhatsApp replaces this) |
| **Kosher Diet** | Small market in India |

### Gaps Analysis

| Gap | Impact | Mitigation |
|-----|--------|------------|
| **No iOS** | ~5% of Indian smartphone users excluded | Low priority - can add in Phase 2 if demand exists |

All other features have parity with or exceed Ollie.
