### When lock/unlock indicator appears:

| Scenario | Icon | Meaning |

### Notes:
- Lock/unlock icons are **informational only** - no tap action on this screen
- To lock/unlock, user must return to Home screen (Screen 4)
- All recipe actions remain enabled regardless of lock status (cook, favorite, modify, add to grocery)
- Lock only affects swap/regeneration behavior on Home screen
